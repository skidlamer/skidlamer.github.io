Instructions

SkidFest is a player aid for the game Krunker To use skidfest you will need to install TamperMonkey browser addon, if you do not have it already. Once installed you can Download Skidfest The hotkey for opening the skidfest settings in game is [ F1 ] alternatively you may use the middle mouse button (scroll wheel)

Sourcecode

Skidfest is open source and MIT licensed. This means you can use it for any project - even commercial projects! Download it, customize it, and share it!
